STACK (REMAKE)

Language: C#, Unity 5

Platform: Android

Stack (Remake) is a 3D video game originally developed by Ketchapp, and recreated using C# and Unity 5 by Ian Chuck-Yin. The objective is to stack the tower as high as possible with tiles, where each successful stack contributes towards points. If the tile is not perfectly placed on top of the stack, a piece of the tile is removed causing the stack to shrink as the game progresses. The game includes a high score system.
