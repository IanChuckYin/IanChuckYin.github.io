ZOMBIE BOYS

Language: Java, Java Swing

Platform: PC

Zombie boys is a 2D shooter video game developed using Java and Java Swing by Ian Chuck-Yin. The objective is to defend your house from incoming zombies. Each zombie earns the player gold to spend in the available in-game shop where the player can purchase stronger weapons. The game includes three levels, an instructions page, and an in-game shop.
